# Sonia Sharman ARG Investigation

This repository documents an ongoing investigation into the Sonia Sharman YouTube channel and related accounts.

I stumbled upon the channel by accident and started looking into it because it appeared to contain ARG-like clues, coded messages, strange comments, and connected accounts. The goal of this project is to preserve the evidence, decode messages, organize the timeline, and track possible connections without losing the order in which discoveries were made.

## Main Document

The investigation notes are kept in:

```text
documentation.md
```

That file contains:

* decoded Morse/code messages
* video and short observations
* comment thread translations
* related account findings
* open questions and theories
* upload timetable
* local image evidence references

## Research Method

All information was found and gathered manually by me while investigating the channel, comments, related profiles, Reddit posts, and other leads.

Codex was only used to help format, organize, decode, and maintain the Markdown documentation in a readable way. It did not independently discover the clues.

## Structure

The main documentation follows discovery order unless a section explicitly says otherwise. This means entries are generally placed in the order I investigated them, not always in the order the videos were originally uploaded.

For actual upload order, see the `Upload Timetable` section at the end of `documentation.md`.

## Notes

This is an investigation log, not a finished explanation. Some connections are confirmed by evidence, while others are marked as theories or open questions.

Any real people, usernames, or accounts mentioned should be treated carefully and respectfully. A connection to the ARG is not proof that someone created it or was involved in it.
